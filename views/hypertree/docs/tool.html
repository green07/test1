  2. 「企業の合併や業務提携等に伴い、企業間を跨っての集計システムの統合や再構築を行いたい」等の今日的な利用企業ニーズに対し「短納期／省資源で（つまり、少ない情報化投資で）処理時間も高速なシステムの実現」の強力なツールとなる。
